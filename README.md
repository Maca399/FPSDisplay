# FPS Display

This script adds a FPS viwer into your Unity game so you can use it to check your FPS so you don't need external tools

---

## Features

able to make the fps into  a interger or not

---

## Requirements

- works with all unity versions
- TextMeshPro

---

## Installation

1. Import the script.
2. Drag the script into your scene.
3. Assign the references.
4. Done!

---

## Inspector Setup

the first veriable is the TextMeashProUGUI so just assine the text into there and if you want the fps to be in decimals just click the desimal box
